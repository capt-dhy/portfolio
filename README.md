# dhyTech Portfolio

A fully responsive personal portfolio website for a Junior Fullstack Developer, showcasing projects, skills, and contact information. The design features a modern dark theme with neon glow effects and smooth animations.

## Features

- **Responsive Design**: Built with Bootstrap 5 and custom CSS to ensure compatibility across mobile, tablet, and desktop devices.
- **Modern UI**: Dark theme (`#0f0f1a`) with purple/cyan accent colors and glassmorphism effects on the navbar.
- **Interactive Elements**:
  - Hover effects on project cards (zoom and lift).
  - Animated skill bars.
  - Glowing buttons and social icons.
- **Sections**:
  - **Hero**: Introduction and profile photo.
  - **About & Skills**: Professional summary and technical proficiency bars.
  - **Projects**: Grid layout showcasing work with tech stack icons.
  - **Footer**: Social media links and copyright info.

## Technologies Used

- **HTML5**
- **CSS3** (Custom variables, Flexbox, Animations)
- **Bootstrap 5.3.3** (Grid system and components)
- **Font Awesome 6.4.0** (Social and tech icons)
- **Google Fonts** (Poppins typeface)

## Setup

1. Clone the repository or download the source files.
2. Ensure `index.html`, `style.css`, and your image assets are in the same directory.
3. Open `index.html` in any modern web browser.

## Customization

- **Colors**: Edit the `:root` variables in `style.css` to change the global color scheme (e.g., `--accent-purple`, `--bg-main`).
- **Content**: Update the text within `index.html` to match your personal details.
- **Images**: Replace `dhy.jpg` and the project image URLs with your own assets.
